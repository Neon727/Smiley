## 🛠️ Setup Requirements

To speak with Smiley, you must have **Ollama** and the **Llama 3** model installed on your machine. 

### Quick Installation:
1. Make sure **Ollama** is downloaded and running in the background.
2. If you don't have the Llama 3 model yet, open your terminal or command prompt and run:
   ollama run llama3
3. Once the model finishes downloading, launch `hello.exe` and Smiley will be waiting.
